# كيفية إعادة تحميل البيانات الافتراضية

إذا لم تظهر الأعداد الصحيحة للاختبارات، اتبع الخطوات التالية:

## الخطوات

### 1. افتح Console في المتصفح
- اضغط `F12` أو
- انقر بزر الماوس الأيمن → "Inspect" → "Console"

### 2. امسح البيانات القديمة
نفذ الأمر التالي في Console:
```javascript
localStorage.clear()
```

### 3. حدث الصفحة
- اضغط `F5` أو
- انقر على زر التحديث في المتصفح

### 4. تحقق من البيانات
بعد التحديث، يجب أن ترى الأعداد الصحيحة:

| التصنيف | عدد الاختبارات |
|---------|----------------|
| العلوم | 0 |
| الرياضيات | 5 |
| التاريخ | 10 |
| الجغرافيا | 15 |
| اللغة العربية | 20 |
| اللغة الإنجليزية | 0 |
| الفيزياء | 5 |
| الكيمياء | 10 |
| الأحياء | 15 |
| الحاسب الآلي | 20 |
| التربية الإسلامية | 5 |
| الفنون | 10 |

## التحقق من البيانات في Console

يمكنك التحقق من البيانات المخزنة بتنفيذ:

```javascript
// عرض التصنيفات
JSON.parse(localStorage.getItem('quiz_app_categories'))

// عرض الاختبارات
JSON.parse(localStorage.getItem('quiz_app_quizzes'))

// حساب عدد الاختبارات لكل تصنيف
const quizzes = JSON.parse(localStorage.getItem('quiz_app_quizzes'));
const counts = {};
quizzes.forEach(q => counts[q.category] = (counts[q.category] || 0) + 1);
console.table(counts);
```

## ملاحظة مهمة

البيانات الافتراضية يتم تحميلها **فقط** عند أول استخدام أو بعد مسح LocalStorage. إذا كانت لديك بيانات قديمة، يجب مسحها أولاً باستخدام `localStorage.clear()`.
